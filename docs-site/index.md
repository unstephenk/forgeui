# ForgeUI

ForgeUI is a CLI that turns **Tokens Studio exports** into **CSS variables** and a **Tailwind preset**.

- Source repo: https://github.com/unstephenk/forgeui
- Getting started: [/getting-started](/getting-started)
- Troubleshooting: [/troubleshooting](/troubleshooting)
- Supported token types: [/supported-token-types](/supported-token-types)

## Mission
Make design-system tokens boringly reliable—one source of truth, deterministic outputs, and zero hand-edited drift.

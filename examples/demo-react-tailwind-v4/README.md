# ForgeUI Demo (React + Tailwind v4)

This is a minimal demo app showing ForgeUI outputs wired into Tailwind v4 CSS-first.

## Run

From repo root:

```bash
# 1) Install CLI deps
npm i

# 2) Generate ForgeUI outputs from the example tokens
cp examples/demo-react-tailwind-v4/tokens.json ./tokens.json
npx tsx src/index.ts sync

# 3) Run the demo app
cd examples/demo-react-tailwind-v4
npm i
npm run dev
```

## What to look for
- Colors come from CSS vars generated by ForgeUI.
- Toggle dark mode by setting `data-theme="dark"` on `<html>`.
